作品

  • Signichat

    简介：Signichat是一个基于图片的社交平台，可以在Web上迅速浏览大图，在图片上可以打标记、做评论，聊天互动，分为教育版和会议版。教育版供教师上传讲义、学生上传作业，教师批注作业及师生互动，是中文版本，目前已经停止运营。会议版供演讲者上传自己的演讲文稿，与参会者互动，是英文版本。

    • 会议版网址：https://www.signichat.com

    主要工作：

    • 参与使用 Backbone + Marionette 重构项目，修复各种遗留 bug，独立开发教育版和会议版
    • 采用 smarty 模板、 LESS、AMD 规范进行开发，实现代码模块化，复用度高
    • 调用 AWS S3 的 API，与后端协同完成 PDF 上传、切分、转换工作
    • 使用 Nodejs 实现实时聊天、实时消息通知，使用 WebRTC 实现视频聊天
    • 实现设计师的响应式要求，对手机用户显示 APP 下载页面，并最大化兼容各种浏览器

    ScienceWall

    ScienceWall基于Signichat开发而来，是为科研人员提供的基于大图浏览的社交平台，实验室与知名的wormimage合作，将wormimage的海量生物大图放到ScienceWall的平台上，提供良好的大图浏览体验和互动性。ScienceWall可以以游客身份访问，只有在进行关键操作时才要求登录，是Signichat的一个轻量化版本，同时也加入了一下特色功能。

    网址：http://www.sciencewall.net

    ScienceWall基本功能的Demo（这个功能与Signichat相似）：

    主要工作：

    • 从Signichat提取出轻量化的版本，搭建ScienceWall的基本框架
    • 实现游客访问，并且在关键操作时要求登录
    • 对于超大图册，在访问前加上小图预览页面
    • 接收客户端信号，实现客户端控制 Web 端图片的切换、滑动、缩放、任意标记等功能
    • 深入了解 OpenSeadragon 工作原理，在实验版中用 WebGL 替代 Canvas2D 实现绘图

    ICOST 2016会议官网

    ICOST 2016是2016年在武汉举行的国际会议，非常荣幸的担任Web Master，负责会务网站的建设、信息发布，并且其他后勤相关事宜。网站现在收录在ICOST会议的history中。

    网址：http://www.icostconference.org/2016/index.html

    主要工作：

    • 搭建会议网站，配上会务组及时更新会务信息，并负责国内注册等后勤事宜。
    • 配合设计师，实现各种布局和动画效果，实现响应式设计，兼容 Retina 屏幕显示
    • 根据IP区分国内外用户，国内用户用优酷的视频链接，国外用户用youtube的视频链接

    WBEdu在线教育

    WBEdu是一个在线教育网站，供国外的老师给国内的学生教学，主要面向儿童。网站提供教师、学生和代理三种角色的课程预约、上课打卡、课后反馈等功能。

    网址：http://www.kaixinabcd.com

    主要工作

    • 对 FullCalendar 深度定制，实现设定时间可选、预定课程、确认上课完成等多种状态
    • 与后台确定 RESTful API，实现学生、教师、代理三种用户的不同权限控制

    GenePlus

    GenePlus是一个医患管理系统，包括咨询、随访和查看报告等功能，目前还在内测中

    网址：内测中，敬请期待

    主要工作

    • 基于 Vue.js 和 SASS，实现医生主页面的整体布局，并完成病人选项卡下的内容
    • 讨论商定可视化的技术方案，基于之前学习的可视化知识提供指导
